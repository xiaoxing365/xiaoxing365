# 小兴365的个人网站

## 网站介绍
这是小兴365的个人网站，主要用于展示个人信息、作品和链接集合。

## 功能特点

1. **个人主页** - 展示个人信息、座右铭和一言动态
2. **关于页面** - 简单的个人介绍
3. **链接集合** - 个人常用链接的整理和搜索
4. **404页面** - 优雅的404错误页面

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript** - 交互功能
- **GitHub Pages** - 网站托管

## 项目结构

```
├── index.html          # 首页
├── about.html          # 关于页面
├── others.html         # 链接集合页面
├── 404.html            # 404页面
├── README.md           # 项目说明文档
├── .gitignore          # Git忽略文件
└── assets/             # 静态资源目录
    ├── CSS/            # CSS样式文件
    ├── JS/             # JavaScript文件
    ├── pictures/       # 图片资源
    └── curs/           # 自定义光标
```

## 本地运行

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/xiaoxing365/xiaoxing365.github.io.git
   ```

2. 使用浏览器打开 `index.html` 文件即可查看网站。

## 部署方式

网站通过 GitHub Pages 部署，将代码推送到 `main` 分支后，GitHub 会自动部署网站。
